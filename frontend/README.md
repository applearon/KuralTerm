# kurual-frontend

To install dependencies:

```bash
bun install
```

Set the environment variables in `.env`:
```
export URL=localhost;
export PORT=3000;
```

To run:

```bash
bun run index.js
```



This project was created using `bun init` in bun v1.0.3. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
